# Amazon Prime Video Content Analytics Dashboard

An interactive Power BI dashboard analyzing Amazon Prime's content library, tracking over 9,600+ titles to uncover insights into content distribution, genres, and historical release trends.

## 📊 Dashboard Preview
![Dashboard Screenshot](Logo.png) 

## 🚀 Key Insights
* **Content Split:** Movies heavily dominate the platform at **80.82%** (7.81K titles) compared to TV Shows at **19.18%** (1.85K titles).
* **Top Genres:** Drama is the single largest genre category with nearly 1,000 titles, followed closely by Comedy.
* **Growth Trend:** Content volume experienced exponential growth starting around the year 2000, peaking heavily toward 2021.

## 🛠️ Tools Used
* **Power BI Desktop:** Data modeling, DAX measures, and dashboard visualization.
* **Power Query:** Data cleaning, handling missing values, and attribute filtering.
* **Dataset:** Raw Amazon Prime video data (`.csv` format included in repo).
